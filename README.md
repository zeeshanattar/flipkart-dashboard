# 1) project title : flipkart sales analytics
flipkart sales analytics : A dynamic, interactive data visualization tool typically designed to give a bussiness team a 360 degree view of marketplace performance and to explore on regional comparison which can suit different attributes , complexity and suitability of ecommerce business platforms.
Short description/ purpose

# 2) Description / purpose 
the flipkart analytics is a visually engaging and analytical power bi report designed to help users explore and compare the sales fluctuation, the sales trend , sales fluctuations in different regions and cities in various product categories , payment modes. The dashboard focuses on highlighting major sales distribution and the marking of the peaks and the downfall over the monthly and quarterly basis. This tool is intended for use by different business analyst for ecommerce platform to understand trend and characteristics of the sales comming from different categories , mark the problem and take the data driven decisions.

# 3) Tech Stack
List of key technologies used to build the dashboard.
The dashboard was built using the following tools and technologies:

Power BI Desktop: Main data visualization platform used for report creation.

Power Query: Data transformation and cleaning layer for reshaping and preparing the data.

DAX (Data Analysis Expressions): Used for calculated measures, dynamic visuals, and conditional logic.

Data Modeling: Relationships established among tables to enable cross-filtering and aggregation.

File Format: .pbix for development and .png for dashboard previews

# 4) Data source
This dashboard is built using sample/synthetic e-commerce data modeled to represent Flipkart-like operations.

Business problem : To mark the reason for the considerable downfall of the sales and the profit in the month of april to july
